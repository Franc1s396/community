# DH人人社区
简易版论坛，仅供萌新参考，大佬勿喷
<br>
目前还在开发中
## 功能
1. jwt+易于扩展的登录方式
   1. 用户名密码
   2. 邮箱
2. 第三方授权登录
   1. github
3. 帖子模块
   1. 帖子,评论,标签-增删改查
   2. 帖子PV统计
   3. 为帖子点赞 
   4. 发布帖子后为粉丝推送
4. 关注模块
   1. 关注用户
   2. 用户关注列表
   3. 用户粉丝列表
   4. 查询共同关注
## 技术栈 
1. SpringBoot  
2. SpringSecurity  
3. Thymeleaf
4. Redis
5. MySQL
6. Mybatis-Plus
7. Knife4j(Swagger)
## 快速运行
1. clone到本地``git clone https://github.com/Franc1s396/community.git``
2. 运行sql脚本到本地数据库
3. 修改配置文件
4. 启动``CommunityApplication``即可运行
