# DH人人社区
简易版论坛,还在开发中.
## 功能
1. 多种方式(用户名密码、邮箱)登录
2. github第三方授权登录
## 技术栈 
1. SpringBoot  
2. SpringSecurity  
3. Thymeleaf
4. Redis
5. MySQL
6. Mybatis-Plus
7. Knife4j(Swagger)
## 快速运行
1. clone到本地``git clone https://github.com/Franc1s396/community.git``
2. 运行sql脚本到本地数据库
3. 修改配置文件
4. 启动``CommunityApplication``即可运行
